# Examples Guide

Bundled examples live in `examples/`.

## `hello.deck`

Minimal deck:

```bash
node dist/cli.js run examples/hello.deck
```

Demonstrates program boundaries and `socialise`.

## `fizzbuzz.deck`

Classic fizzbuzz.

Demonstrates:

- for loops
- conditionals
- modulo
- string and number output

## `fibonacci.deck`

Iterative Fibonacci.

Demonstrates:

- functions
- returns
- while loops
- variable reassignment

## `strategy.deck`

Small strategy narrative.

Demonstrates:

- booleans
- logical operators
- if / else-if / else flow

## `performance-review.deck`

Review narrative generator.

Demonstrates:

- multi-branch function returns
- function calls with different inputs

## `offsite.deck`

Agenda pipeline.

Demonstrates:

- array literals
- pushing onto arrays
- indexed array access

## `quarterly-planning.deck`

Loop-until-ready workflow.

Demonstrates:

- `circle back until`
- boolean state changes
- guarded loops

## `data-room.deck`

Collections showcase.

Demonstrates:

- array index assignment
- array removal
- map/filter/sort
- object/dictionary/map literals
- object lookup and assignment
